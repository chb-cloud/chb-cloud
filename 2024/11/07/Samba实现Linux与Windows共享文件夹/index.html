Samba实现Linux与Windows共享文件夹（一）——部署Ubuntu服务端

一、背景知识介绍

1、Samba 介绍

Samba 是一套在 Linux 和 UNIX 系统上实现 SMB 网络协议的软件。它使得 Linux、Unix 与 Windows 系统之间能够实现文件和打印机的共享，以及进行用户认证等功能。Samba 可以看作是在非 Windows 操作系统上模拟 Windows 文件共享服务的工具。

2、SMB（Server Message Block）介绍

SMB 是一种网络文件共享协议，主要用于在计算机之间共享文件、打印机、串口等资源。它允许不同操作系统的计算机在网络上进行文件和资源的共享与访问。
